1. Instalar Node.js download em https://nodejs.org/pt-br/download/
2. Instalar MongoDB download em https://fastdl.mongodb.org/win32/mongodb-win32-x86_64-2012plus-4.2.8-signed.msi
3. Executar o "Node.js Command Prompt"
4. Executar o comando "npm i -d nodemon"
5. Executar o comando "npm i bcrypt body-parser connect-mongo dialog-polyfill dotenv ejs express express-flash express-session html2canvas mongodb mongoose multer passport passport-local request"
6. Executar o comando "npm start"
7. Acessar o projeto via browser utilizando "localhost:8081" como endereço
